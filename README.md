## ✨ Family Docs v2.0

Aceasta este soluția perfectă pentru gestionarea documentelor familiei. Este o platformă foarte modernă, cu un design special, numit Glassmorphism, care asigură sincronizarea în siguranță a documentelor importante.

---

## 🚀 CARACTERISTICI IMPORTANTE

### 🔒 Securitate de nivel militar

Validare utilizând codul numeric personal și algoritmi speciali pentru verificarea identității.

### 🌫️ Interfață cu design Glassmorphism

Design simplu, în modul întunecat, cu efecte de estompare și transparență.

### 🔔 Alerte inteligente

Notificări automate pentru documentele care urmează să expire în curând.

### 👨‍👩‍👧‍👦 Hub pentru familie

Administrează membrii, rolurile (Părinte/Copil) și cererile de acces în timp real.

---

## 🛠️ TECH STACK

## ⚡ INSTALARE RAPIDĂ

### 1. Clonează depozitul

```bash
git clone https://github.com/VasiBestia/Proiect_ISP.git
```
### 2. Configurează mediul

Importă proiectul în IntelliJ IDEA ca proiect Maven.

### 3. Execută

Pornește clasa principală Spring Boot și accesează:

```bash
localhost:8080
```
